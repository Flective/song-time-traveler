## WHAT IS THIS?

**Fiat Musica v7.0** is the complete operating system for managing the Platonic Album Model - a revolutionary approach to music in the age of AI.

This is NOT just a catalog app. This is:
- **Virtual Song Management** (compositions as Platonic ideals)
- **Persona System** (recurring artistic identities)
- **Living Archive** (temporal storytelling)
- **My Album Builder** (user co-curation)
- **Founding Member Portal** (investment & governance)
- **Documentation Library** (intellectual property organization)
- **Complete Analytics** (performance intelligence)
- **Band Operations** (Of Broken Light HQ)
- **Artist Outreach** (pitching to established artists)
- **Protosong Workshop** (cassette archive development)

## QUICK START

### Installation

```bash
# Navigate to outputs directory
cd /mnt/user-data/outputs

# Run the app
streamlit run fiat_musica_v7_platonic.py
```

### First Launch

On first launch, the app automatically:
1. Creates 4 default personas (Meridian Quartet, Lumen Vox, Of Broken Light, Neon Room)
2. Initializes empty data files
3. Sets up the complete tab structure

## WORKFLOWS

### Workflow 1: New Song from Scratch
1. Go to Virtual Songs → Add New Virtual Song
2. Enter title, year, themes, lyrics, ratings
3. Go to Manifestations sub-tab → Add manifestation
4. Select persona, add production details
5. Go to Living Archive → Add Genesis chapter
6. Set Strategic Positioning
7. Add initial review

### Workflow 2: Cassette to Virtual Song
1. Go to Protosongs → Archive New Protosong
2. Document tape details, transcribe lyrics
3. Log evolution as you develop
4. When ready, click "Graduate to Virtual Song"
5. Virtual Song created automatically with Genesis chapter

### Workflow 3: Building My Album
1. Go to My Album Builder
2. Name your album
3. For each song, select your preferred manifestation
4. Set track order (optional)
5. Save configuration

### Workflow 4: Band Operations
1. Band HQ → Roster → Update positions
2. Band HQ → Setlist → Reorder tracks
3. Band HQ → Rehearsals → Log practice
4. Band HQ → Performances → Schedule show

### Workflow 5: Artist Pitch
1. Artist Outreach → Add Target Artist
2. Select songs that fit
3. Explain why the match works
4. Log initial research interaction
5. Update status as pitch progresses

## TROUBLESHOOTING

### App won't start
```bash
# Check you're in the right directory
pwd

# Should show: /mnt/user-data/outputs
# If not:
cd /mnt/user-data/outputs

# Run again
streamlit run fiat_musica_v7_platonic.py
```

### Data not saving
- Check file permissions in working directory
- Look for error messages in terminal
- Try Settings → Reload All Data

### Features not working
- Make sure you've added base data (virtual songs, personas)
- Some features require data dependencies (e.g., My Albums needs virtual songs)
